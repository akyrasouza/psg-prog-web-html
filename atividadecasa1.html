Criando servidor web Arduino e controlando coisas via WiFi - Tutorial

A IOT está à nossa volta há algum tempo e quase revolucionou a maneira como as máquinas se comunicam entre si.

Como ponto de partida, aqui está um tutorial de projeto simples sobre a criação de um servidor da web Arduino que o ajudará a abrir as portas para IOT.

Neste tutorial, hospedaremos um servidor web simples, conforme mostrado na imagem acima.

COMPONENTES NECESSÁRIOS:

  1. Arduino Uno
  2. Módulo WiFi ESP8266
  3. LEDs
  4. Fios de conexão
  5. Conversor lógico direcional de pássaros
  6. Celular ou laptop ou outros dispositivos habilitados para WiFi

O que é HTTP 3 e qual a diferença?

O HTTP 3 é a nova versão do protocolo HTTP, o suporte ao HTTP/3 já existe desde setembro de 2019 na versão de testes do Chrome, embora ainda não esteja ativado por padrão em qualquer navegador, em 2020 começou a ter o suporte não-padrão nas versões estáveis do Chrome e Firefox e ele já pode ser ativado. Já para o navegador da Apple, o Safari, o suporte experimental para HTTP/3 foi adicionado ao Safari Technology Preview em 8 de abril de 2020.
